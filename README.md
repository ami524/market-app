# フリーマーケットアプリ
## 環境構築
Dockerビルド
1. git clone git@github.com:ami524/market-app.git
2. docker-compose up -d --build

Laravel環境構築
1. docker-compose exec php bash
2. composer install

## 使用技術
* PHP 7.4.9
* Laravel 8.83.8
* MySQL 8.0.26

## URL
* 開発環境：http://localhost/
* phpMyAdmin：http://localhost:8080/
